This document explains how to load docker images needed for symbIoTe Enablers. 
This is convenient if you don't want to use network for downloading docker 
images from docker hub.

=======================
Loading docker images
=======================

In order to load images run load_images.sh script or run manually following 
command for each image in images directory:

$ docker load -i images/imageName.tar

=======================
